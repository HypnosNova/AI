### 符号约定

$m​$ ：训练集的规模，即训练样本的数量

$n_x$ ：特征向量x的维度

$(x,y)$ ：一个单独的样本

$(x^{(i)},y^{(i)})$ ：在m中第i个样本，m可以表示为{$(x^{(1)},y^{(1)}),(x^{(2)},y^{(2)}),(x^{(3)},y^{(3)}),...(x^{(m)},y^{(m)})$}

$m_{train}$ : 训练集

$m_{test}$ : 测试集

$X=[x^{(1)},x^{(2)},x^{(3)},...,x^{(m)}]$ ：一个$n_x*m$ 的矩阵
